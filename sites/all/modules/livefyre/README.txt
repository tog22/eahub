The Livefyre module allows for Livefyre comments to be inserted into your page.
Currently, they can be inserted into a node ('Content Area') or a Block
('Block').

Simple Installation
1. Download and install the Livefyre module into your modules directory
(sites/all/modules).
2. Enable the module at http://YOUR-SITE-NAME.com/admin/modules
*(SEE NOTE BELOW)
3. Register your site at http://livefyre.com
4. Visit http://YOUR-SITE-NAME.com/admin/config/livefyre and set your
configuration options depending on what you registered with on Livefyre.

This module inserts Livefyre comments into your site via javascript. Currently,
this module allows 2 choices where you can place your Livefyre comments:

    -in a node's 'Content Area'
    -in a 'Block'

* PLEASE NOTE:
This module does not disable the default Drupal 'Comment Module'.  To disable
the Drupal Comment module, simply visit http://YOUR-SITE-NAME.com/admin/modules
and uncheck the checkbox for the Comment Module.  Click 'Save configuration'
at the bottom of the page and you're done!
